## 소개

CoBookE는 북마크 관리 App입니다.
CoBookE는 북마크를 관리하는 데에 있어 차별화된 강점을 가지고 있습니다.

1. 코스 별, 항목 별 관리 가능
1. URL 동시 공유 기능
1. 다른 App에서 저장
1. 각종 Setting
