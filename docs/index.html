Course Bookmark with Efficiency

CoBookE 는 북마크를 관리하는 데에 있어 차별화된 강점을 가지고 있습니다.

코스 별 관리

데이트나 여행 등의 일정을 전반적으로 스케쥴링하기에 적합합니다. 공통된 검색어는 추출이 가능합니다.

항목 별 관리

파일과 폴더 형식으로 개별적인 항목을 관리합니다. 이동, 복사, Navigation bar 와 같은 편의를 제공합니다.

코스 공유

코스 내 모든 URL 항목들을 한 번의 터치로 공유할 수 있습니다. 그룹이나 개별 항목도 공유할 수 있습니다.

폴더 공유

코스와 마찬가지로 폴더 내 모든 항목을 한꺼번에 공유할 수 있습니다. 폴더 Level은 들여쓰기로 구분합니다.

다른 App에서 저장하기

다른 App에서도 공유 기능으로 손쉽게 저장할 수 있습니다. 저장 위치 및 실행 여부를 선택할 수 있습니다.

각종 설정

검색 엔진, 홈페이지, 초기 선택 Tab, 삭제 확인 여부 등을 선택할 수 있고, 전체 삭제를 할 수 있습니다.

Open Source Library

사용한 Open Source Library 중 3가지 대표 Library입니다.

Realm

모바일 데이터베이스 Library 입니다. 짧고 간결한 코드로 DB를 손쉽게 관리한다는 큰 장점이 있습니다. 코스, 폴더 등 상하 구조를 관리하는데 큰 도움이 되었습니다.

Jsoup

HTML Parsing Library 입니다. url 주소만 있으면 select 메서드를 사용하여 쉽게 사용이 가능합니다. 간단한 Pattern 적용을 통해 원하는 tag에 접근할 수 있습니다.

Glide

이미지 Loading Library 입니다. 메서드 체인 방식으로 직관적인 적용이 가능합니다. thumbnail 함수를 사용했고, listener를 붙여 ProgressBar의 Visibility를 처리했습니다.
