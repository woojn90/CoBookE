CoBookE

Course Bookmark with Efficiency App을 사용하면
당신의 북마크를 편리하게 관리할 수 있습니다.

  • Github Page
  • Learn More

CoBookE는 단순한 북마크 관리 App이지만,
기존 App과는 분명한 차이점이 있습니다.

그룹화 관리
(Category Management)

코스 별, 항목 별 관리가 가능합니다. 처음부터 그룹화가 된 상태라면 코스를 사용하고, 개별 항목에서 시작하여 폴더로 관리하고자 한다면 항목을 사용합니다.

검색어 지정 및 자동 검색
(Auto search using specified word)

해당 그룹에 알맞은 검색어를 미리 정해두고 이를 활용하여 검색합니다. 이전에 보았던 웹 페이지를 다시 찾기하기 위해 기억 속 검색어를 되새길 필요가 없습니다. 이러한 기능들은 모두 CoBookE 어플 안에서 동작합니다.

다른 어플에서 저장하기
(Save bookmark from another app)

다른 어플에서 보던 내용도 공유 기능을 사용하여 손쉽게 저장할 수 있습니다. 저장 시 위치를 지정할 수 있고, 본인 선택에 따라 웹 서핑을 지속하거나 CoBookE 어플을 실행할 수 있습니다.

저장 항목 공유하기
(Share saved items)

분류하여 저장해둔 항목들을 원하는 범위만큼 공유할 수 있습니다. 북마크 주소가 제목, 설명과 함께 정리되어 공유됩니다.

  • 이동하려면 한 번 누름
  • 수정하려면 두 번 누름
  • 삭제하려면 길게 누름
  • 2개 컬럼으로 변경
  • 3개 컬럼으로 변경
